# S2_Project
 Survival game developed with Unity




# 게임 소개

이 게임은 기본적으로 로그라이크의 구성을 계획중입니다.
던전(미로)를 빠져나오는 과정에서 선택지에 따라 뱀서라이크와 같은 생존이나, 퍼즐 등을 거치면서 아이템을 획득하여 클리어하는 형식의 게임을 계획중입니다.




# 게임 개요

로그라이크 + 서바이벌 + 퍼즐
